# 🎓 StudyAI Pro

AI-Powered Multi-Agent Learning Assistant using RAG, Memory, Analytics, and Personalized Tutoring.

## 🚀 Key Features

- Multi-Agent Architecture
- Retrieval-Augmented Generation (RAG)
- AI Tutor
- Quiz Generator
- Flashcard Generator
- Revision Notes Generator
- Learning Roadmaps
- Learning Analytics Dashboard
- User Memory
- Secure PDF Processing
