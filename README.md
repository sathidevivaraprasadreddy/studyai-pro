# StudyAI Agent

## Overview

StudyAI Agent is an AI-powered learning assistant that helps students study from PDF documents. The system uses Retrieval-Augmented Generation (RAG), memory, and multiple specialized agents to generate notes, quizzes, flashcards, study roadmaps, and personalized tutoring support.

## Problem Statement

Students often struggle to extract important information from lengthy academic documents. Manual note-taking and revision are time-consuming and inefficient.

## Solution

StudyAI Agent automates the learning process by:

* Uploading and processing PDF documents
* Generating concise study notes
* Creating flashcards for revision
* Generating quizzes and assessments
* Providing AI tutoring assistance
* Creating personalized learning roadmaps

## Features

* PDF Upload and Processing
* Vector Search using FAISS
* Multi-Agent Architecture
* Flashcard Generation
* Quiz Generation
* Study Planning
* Learning Roadmaps
* Student Progress Tracking

## Tech Stack

* Python
* Flask
* FAISS
* Sentence Transformers
* Google Gemini API
* HTML/CSS/JavaScript
